# 快速开始指南

## 🚀 5分钟上手

### 第一步：安装依赖

```bash
pip install -r requirements.txt
```

### 第二步：获取今日验证码

**方法1（推荐）**：双击运行 `获取验证码.bat`
- 打开图形化界面
- 点击「复制验证码」按钮
- 自动复制到剪贴板

**方法2**：命令行运行
```bash
python password_generator_gui.py
```

**方法3**：传统命令行版本
```bash
python password_generator.py
```
或双击运行 `生成验证码.bat`

会显示类似以下内容：
```
今天的验证码是: 379970
```

复制这6位数字。

### 第三步：启动软件

```bash
python app.py
```

### 第四步：输入验证码

在弹出的验证对话框中：
1. 输入刚才复制的6位数字验证码
2. 点击"✓ 验证"按钮
3. 验证通过后即可使用软件

## 💡 常用操作

### 提取联系人
1. 切换到"📋 联系人提取"标签页
2. 粘贴OCR识别后的文本
3. 点击"🔧 提取联系人"
4. 点击"📤 用到发送工具"同步到发送页面

### 批量发送
1. 切换到"📤 批量发送"标签页
2. 输入发送内容（文字和/或图片）
3. 粘贴联系人名单（每行一个）
4. 调整延时配置（可选）
5. 点击"▶ 开始运行"

### 控制发送
- **F4**：暂停/继续发送
- **ESC**：紧急停止

## ⚙️ 常见问题

**Q: 验证码每天都需要重新获取吗？**
A: 是的，验证码每天自动变更，需要重新运行 `password_generator.py` 获取。

**Q: 可以在多台电脑上使用同一个验证码吗？**
A: 可以，同一天内所有设备使用相同的验证码。

**Q: 忘记密码怎么办？**
A: 重新运行 `python password_generator.py` 即可查看今天的验证码。

**Q: 验证码输错了怎么办？**
A: 会有错误提示，清空输入框后重新输入即可。
