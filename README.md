# 企业微信批量发送工具

## 📖 简介

这是一个基于 Python 的企业微信批量消息发送工具，可以自动搜索联系人并发送相同的消息，适用于需要向多人发送通知、公告等场景。

## ✨ 功能特点

- ✅ **自动化发送**：自动搜索联系人、打开聊天窗口、发送消息
- ✅ **智能定位**：通过姓名搜索定位，不受联系人置顶影响
- ✅ **批量处理**：支持一次性发送给多个联系人
- ✅ **灵活配置**：可自定义发送延时，适应不同网络环境
- ✅ **暂停/继续**：支持 F4 热键暂停和继续
- ✅ **紧急停止**：支持 ESC 热键立即停止
- ✅ **线程安全**：稳定的日志系统，不会卡死界面

## 📦 安装依赖

```bash
pip install -r requirements.txt
```

或者手动安装：

```bash
pip install pyperclip pyautogui keyboard
```

## 🚀 使用方法

### 1. 启动程序

```bash
python main.py
```

### 2. 填写信息

- **发送内容**：输入要发送的消息文本
- **联系人名称**：每行输入一个联系人姓名（支持空白行分隔）
- **延时配置**（可选）：
  - 粘贴后：粘贴消息后的等待时间（默认 0.2 秒）
  - 发送后：发送消息后的等待时间（默认 0.4 秒）
  - 切换：搜索联系人时的等待时间（默认 0.3 秒）

### 3. 开始发送

1. 确保企业微信已经登录并运行
2. 点击 **"开始运行"** 按钮
3. 脚本会自动激活企业微信窗口
4. 3 秒倒计时后开始发送
5. 发送完成后自动关闭企业微信

## ⌨️ 快捷键

| 快捷键 | 功能 |
|--------|------|
| F4 | 暂停 / 继续发送 |
| ESC | 紧急停止发送 |

## 💡 使用示例

### 示例 1：发送通知给 5 个人

**发送内容：**
```
你好，明天上午9点开会，请准时参加。
```

**联系人名称：**
```
张三
李四
王五
赵六
钱七
```

点击"开始运行"，脚本会依次发送给这 5 个人。

### 示例 2：分组发送（使用空行分隔）

**联系人名称：**
```
张三
李四
王五

赵六
钱七
```

空行会被自动忽略，实际发送给 5 个人。

## ⚙️ 高级配置

### 调整延时参数

如果发送过程中出现问题（如消息未发送成功、联系人未找到等），可以尝试增加延时：

- **电脑较慢**：将所有延时增加 0.2-0.5 秒
- **网络较慢**：增加"切换"延时到 0.5-1.0 秒
- **快速发送**：减少延时至 0.1 秒（不推荐，可能不稳定）

### 常见问题

**Q: 发送时找不到联系人？**
A: 确保联系人姓名与企业微信中显示的完全一致（包括空格、特殊字符）。

**Q: 发送中途想停止？**
A: 按 ESC 键即可紧急停止。

**Q: 想临时暂停？**
A: 按 F4 键暂停，再次按 F4 继续。

**Q: 可以发送图片吗？**
A: 当前版本仅支持文本消息。

**Q: 可以给不同的人发送不同的消息吗？**
A: 当前版本所有人发送相同消息，如需不同消息需要修改代码。

## ⚠️ 注意事项

1. **首次使用建议测试**：先用 2-3 个联系人测试，确认一切正常后再批量使用
2. **保持窗口焦点**：运行过程中不要切换窗口，确保企业微信保持激活状态
3. **姓名准确性**：联系人姓名必须与企业微信中显示的完全一致
4. **权限要求**：脚本需要控制键盘和剪贴板，运行时可能会被杀毒软件拦截，请添加信任
5. **责任声明**：请勿用于发送垃圾消息或骚扰他人，合理使用本工具
