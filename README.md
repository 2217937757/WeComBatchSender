# 企业微信批量发送工具

## 📖 简介

这是一个基于 Python 的企业微信批量消息发送工具，可以自动搜索联系人并发送相同的消息。配套提供**联系人提取工具**，通过 QQ/微信长截图快速提取大量联系人名单。

**适用场景**：需要向多人发送通知、公告、提醒等消息。

---

## ✨ 功能特点

### 批量发送工具 (main.py / app.py)
- ✅ **自动化发送**：自动搜索联系人、打开聊天窗口、发送消息
- ✅ **智能定位**：通过姓名搜索定位，不受联系人置顶影响
- ✅ **批量处理**：支持一次性发送给多个联系人
- ✅ **灵活配置**：可自定义发送延时，适应不同网络环境
- ✅ **暂停/继续**：支持 F4 热键暂停和继续
- ✅ **紧急停止**：支持 ESC 热键立即停止
- ✅ **线程安全**：稳定的日志系统，不会卡死界面
- ✅ **图文混合**：支持文字、图片、图文混合发送

### 联系人整理工具 (contact_cleaner.py)
- ✅ **智能提取**：从 QQ/微信长截图中自动识别并提取联系人
- ✅ **时间戳识别**：支持多种格式（14:30、昨天、24分钟前、03/18等）
- ✅ **企业微信适配**：自动识别三行布局（联系人+消息预览+时间）
- ✅ **符号清理**：自动删除 @微信、@... 等多余标记
- ✅ **保持原序**：不去重、不排序，保留原始顺序
- ✅ **一键复制**：整理完成后自动复制到剪贴板

## 🚀 快速开始

### 方式一：使用集成版（推荐）⭐

**一个工具搞定所有事情！**

```bash
python app.py
```

**功能：**
- 📋 **联系人提取**：从长截图中提取联系人
- 📤 **批量发送**：自动搜索并发送消息
- 🖼️ **图片发送**：支持单张/多张图片发送
- 🎨 **图文混合**：一条消息同时包含文字和图片
- 🔄 **无缝衔接**：提取后直接用到发送工具

### 方式二：使用独立工具

### 第一步：安装依赖

```bash
pip install pyperclip pyautogui keyboard Pillow pywin32
```

### 第二步：提取联系人名单（1000人约15分钟）

**方法：使用 QQ/微信长截图（推荐）**

1. **长截图**
   - 打开企业微信联系人列表
   - 按 `Ctrl + Alt + A` (QQ) 或 `Alt + A` (微信)
   - 点击“长截图”，滚动截取全部联系人

2. **提取文字**
   - 右键图片 → “识别图中文字”
   - 复制所有识别出的文字

3. **智能整理**
   ```bash
   python contact_cleaner.py
   ```
   - 粘贴文字 → 点击“整理提取”
   - 自动得到干净的联系人列表

📝 **详细教程**：查看 [QQ_SCREENSHOT_GUIDE.md](QQ_SCREENSHOT_GUIDE.md)

### 第三步：批量发送消息

```bash
python main.py
```

1. **填写信息**
   - 发送内容：输入要发送的消息
   - 联系人名称：粘贴从整理工具得到的名单（每行一个）
   - 延时配置：根据电脑性能调整（默认即可）

2. **开始发送**
   - 确保企业微信已登录
   - 点击“开始运行”
   - 脚本自动激活企业微信并开始发送
   - 完成后自动关闭企业微信

⌨️ **快捷键**：
- `F4` - 暂停/继续
- `ESC` - 紧急停止

## 💡 使用示例

### 示例1：发送纯文字消息

**发送内容：**
```
你好，明天上午9点开会，请准时参加。
```

**联系人名称：**
```
张三
李四
王五
```

点击“开始运行”，脚本会依次发送给这3个人。

### 示例2：发送纯图片消息

1. 点击“📁 选择图片”按钮
2. 选择要发送的图片（支持多选）
3. 输入联系人名称
4. 点击“▶ 开始运行”

💡 **提示**：不输入文字，只选择图片，即可发送纯图片消息

### 示例3：发送图文混合消息

**发送内容：**
```
这是本周的工作报告，请查收。
```

**选择图片：**
- 点击"📁 选择图片"
- 选择 report1.png, chart.jpg 等多张图片

**联系人名称：**
```
张三
李四
```

点击"开始运行"，每条消息都会包含文字和多张图片（在一条消息中）。

💡 **发送顺序说明**：程序会先粘贴所有图片，再粘贴文字内容，最后统一发送。这样可以确保图片和文字在同一条消息中，显示效果更佳。

### 示例4：从长截图提取1000个联系人

1. QQ长截图截取全部联系人
2. 右键识别文字并复制
3. 运行 `python contact_cleaner.py`
4. 粘贴 → 整理提取 → 得到干净名单
5. 复制到 main.py 中使用

⏱️ **总耗时**：约10-15分钟

## ⚙️ 高级配置

### 调整延时参数

如果发送过程中出现问题，可以尝试增加延时：

- **电脑较慢**：将所有延时增加 0.2-0.5 秒
- **网络较慢**：增加“切换”延时到 0.5-1.0 秒
- **快速发送**：减少延时至 0.1 秒（不推荐，可能不稳定）

### 联系人名单维护

**问题**：有1000+联系人，如何快速获取？

**解决方案**：使用 QQ/微信长截图 + 智能整理工具

1. **长截图提取**：详见 [QQ_SCREENSHOT_GUIDE.md](QQ_SCREENSHOT_GUIDE.md)
2. **智能整理**：运行 `python contact_cleaner.py`
3. **自动去重**：工具会自动过滤、去重、排序

⏱️ **效率**：1000个联系人约需10-15分钟

### 常见问题

**Q: 如何获取1000+联系人名单？**  
A: 使用 QQ/微信长截图 + contact_cleaner.py 整理，详见 [QQ_SCREENSHOT_GUIDE.md](QQ_SCREENSHOT_GUIDE.md)

**Q: 发送时找不到联系人？**  
A: 确保联系人姓名与企业微信中显示的完全一致（包括空格、特殊字符）。

**Q: 发送中途想停止？**  
A: 按 ESC 键即可紧急停止。

**Q: 想临时暂停？**  
A: 按 F4 键暂停，再次按 F4 继续。

**Q: 可以发送图片吗？**  
A: ✅ 支持！可以选择多张图片，支持纯图片发送或图文混合发送。发送时会先粘贴图片，再粘贴文字，最后统一发送为一条消息。

**Q: 可以给不同的人发送不同的消息吗？**  
A: 当前版本所有人发送相同消息，如需不同消息需要修改代码。

**Q: 为什么不用 OCR 识别？**  
A: QQ/微信自带的文字识别准确率更高、速度更快，无需安装额外软件。

## ⚠️ 注意事项

1. **首次使用建议测试**：先用 2-3 个联系人测试，确认一切正常后再批量使用
2. **保持窗口焦点**：运行过程中不要切换窗口，确保企业微信保持激活状态
3. **姓名准确性**：联系人姓名必须与企业微信中显示的完全一致
4. **权限要求**：脚本需要控制键盘和剪贴板，运行时可能会被杀毒软件拦截，请添加信任
5. **隐私保护**：提取的联系人名单请妥善保管，不得用于非法用途
6. **责任声明**：请勿用于发送垃圾消息或骚扰他人，合理使用本工具
