# 企业微信批量发送工具 (WeCom Batch Sender)

一款功能强大、模块化设计的 Python 自动化工具，专为解决企业微信（WeCom）高频消息发送需求而生。它集成了**智能联系人提取**与**自动化图文混发**功能，通过模拟人工操作实现高效、准确的批量触达。

---

## 🌟 项目亮点

- **🚀 模块化架构**：采用 `core`（核心逻辑）、`gui`（界面交互）、`utils`（工具函数）分层设计，代码清晰，易于二次开发与维护。
- **📋 智能联系人解析**：内置算法可从 QQ/微信长截图识别的杂乱文本中，精准提取并去重联系人名称。
- **🖼️ 高级图文引擎**：支持纯文字、单张/多张图片以及“文字+多图”的混合消息发送，确保接收端显示效果一致。
- **⚙️ 精细化控制**：提供粘贴、发送、切换三个维度的延时配置，完美适配不同性能电脑与网络环境。
- **🛡️ 稳定可靠**：基于线程安全的日志系统与异常捕获机制，支持 `F4` 暂停/继续与 `ESC` 紧急停止。

## 🛠️ 技术栈与依赖

本项目基于 **Python 3.x** 开发，主要依赖以下库：

| 库名 | 用途 |
| :--- | :--- |
| `tkinter` | 构建跨平台的图形用户界面 (GUI) |
| `pyautogui` | 模拟键盘按键与鼠标操作，实现自动化交互 |
| `pyperclip` | 跨平台剪贴板操作，用于快速复制文字内容 |
| `keyboard` | 监听全局热键（如 F4 暂停、ESC 停止） |
| `Pillow (PIL)` | 图像处理，负责将 PNG/JPG 等格式转换为剪贴板兼容的 BMP 格式 |
| `pywin32` | Windows 专属 API 调用，实现底层剪贴板数据写入 |

**安装命令：**
```bash
pip install -r requirements.txt
```

## 💡 使用场景与流程

### 1. 准备联系人名单
如果你有一份包含大量联系人的长截图（例如从企业微信或微信群导出）：
1. 进入程序的 **“📋 联系人提取”** 标签页。
2. 将 OCR 识别后的原始文本粘贴到输入框。
3. 点击 **“🔧 提取联系人”**，系统会自动过滤时间戳、表情符号并去重。
4. 点击 **“📤 用到发送工具”**，名单将自动同步至发送页面。

### 2. 配置发送任务
切换到 **“📤 批量发送”** 标签页：
*   **发送内容**：输入通知文字，或通过 **“📁 选择图片”** 添加附件。
*   **延时配置**：根据电脑反应速度调整。建议初始值为：粘贴 `0.2s`，发送 `0.4s`，切换 `0.3s`。

### 3. 执行与监控
点击 **“▶ 开始运行”** 后，请保持企业微信窗口处于激活状态。你可以通过下方的 **“运行日志”** 实时观察进度。

## ⚠️ 重要注意事项

1.  **权限与安全**：由于涉及自动化控制，杀毒软件可能会误报。请将生成的 `.exe` 文件或 Python 环境加入白名单。
2.  **姓名匹配**：自动化搜索依赖于联系人姓名的**完全匹配**。如果对方设置了备注，请输入备注名。
3.  **环境要求**：目前仅支持 **Windows** 操作系统（因依赖 `pywin32` 进行图片剪贴板操作）。
4.  **合规使用**：请严格遵守企业微信使用规范，避免短时间内高频发送导致账号受限。
