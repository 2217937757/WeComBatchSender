## 📋 完整流程

### **步骤1：使用QQ/微信长截图**

#### 方法A：QQ长截图（推荐）
1. 打开企业微信联系人列表
2. 按 `Ctrl + Alt + A` 开始截图
3. 点击工具栏上的 **"长截图"** 按钮（或按 `C` 键）
4. 缓慢向下滚动鼠标滚轮
5. 直到截取完所有联系人
6. 点击"完成"或按 `Enter`

#### 方法B：微信长截图
1. 打开企业微信联系人列表
2. 按 `Alt + A` 开始截图
3. 点击 **"长截图"** 按钮
4. 向下滚动截取
5. 完成后保存或复制

---

### **步骤2：提取图中文字**

#### 方法A：QQ自带文字识别（最快）⭐
1. 长截图完成后，**不要关闭**
2. 右键点击图片 → 选择 **"识别图中文字"**
3. 等待识别完成
4. 点击 **"复制全部文字"**
5. 粘贴到文本文件或直接使用

#### 方法B：微信文字识别
1. 将长截图保存到桌面
2. 用微信扫一扫打开图片
3. 点击 **"提取文字"**
4. 复制所有文字

#### 方法C：在线OCR工具
如果QQ/微信没有文字识别功能：
1. 访问 https://ocr.wdku.net/ （免费在线OCR）
2. 上传长截图
3. 选择"中文识别"
4. 复制识别结果

---

### **步骤3：使用整理工具**

1. **运行整理工具**
   ```bash
   python contact_cleaner.py
   ```

2. **粘贴文字**
   - 将从QQ/微信复制的文字粘贴到"原始文本"框

3. **点击“整理提取”**
   - 智能识别时间戳（14:30、昨天、24分钟前、03/18等）
   - 自动提取联系人（企业微信布局：时间戳上两行）
   - 清理多余符号（@微信、@...等）
   - 保持原始顺序（不去重、不排序）
   - 自动复制到剪贴板

4. **检查结果**
   - 查看"整理结果"框
   - 确认数量是否正确
   - 如有错误可手动编辑

5. **复制到发送工具**
   - 打开 `main.py` 批量发送工具
   - 粘贴到"联系人名称"输入框
   - 开始发送！

## ⚙️ 提高成功率技巧

### 1. 截图前准备
- ✅ **放大字体**：在企业微信设置中调大字体
- ✅ **简化界面**：关闭侧边栏，只显示联系人列表
- ✅ **清晰背景**：使用浅色主题
- ✅ **稳定滚动**：滚动时保持匀速，不要太快

### 2. 分批处理（推荐）
如果联系人太多（>500人），建议分批：

```
第1批：A-C 开头的联系人
第2批：D-F 开头的联系人
第3批：G-I 开头的联系人
...
```

每批处理后：
- 单独保存为一个文件
- 最后合并所有文件
- 运行去重清理

### 3. 验证准确性
- 数一下企业微信显示的总人数
- 对比提取到的联系人数量
- 抽查几页确认没有遗漏

## 🔧 常见问题

### Q1: QQ长截图无法识别文字？

**解决：**
- 确保QQ是最新版本
- 或者使用微信长截图
- 或使用在线OCR工具

### Q2: 识别出很多乱码？

**解决：**
- 我们的整理工具会自动过滤
- 检查截图是否清晰
- 尝试调整企业微信字体大小

### Q3: 漏掉了一些联系人？

**解决：**
- 检查长截图是否完整
- 可能滚动太快导致部分未截取
- 可以针对遗漏区域重新截图

### Q4: 有英文或数字混入？

**解决：**
- 整理工具会自动过滤非中文
- 只保留2-4个中文字符的姓名
- 手动检查特殊情况

### Q5: 如何知道提取完整了？

**解决：**
```
方法1：查看企业微信底部显示的人数
方法2：按字母分段提取，确保覆盖全部
方法3：随机抽查几个联系人确认存在
```
