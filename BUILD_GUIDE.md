# 打包说明文档

## 📦 打包脚本说明

本项目提供了两个打包脚本，分别用于不同的打包需求。

---

## 🔧 方式一：只打包主程序

### 适用场景
- 只需要主程序exe
- 用户已有其他方式获取验证码
- 快速测试打包

### 使用方法
```bash
双击运行：build.bat
或命令行：build.bat
```

### 生成文件
```
dist/
└── 企业微信批量发送工具.exe
```

---

## 🚀 方式二：打包全部工具（推荐）

### 适用场景
- 需要完整分发给其他用户
- 包含主程序和验证码工具
- 正式版本发布

### 使用方法
```bash
双击运行：build_all.bat
或命令行：build_all.bat
```

### 生成文件
```
dist/
├── 企业微信批量发送工具.exe      # 主程序
├── 获取验证码.exe                 # GUI版验证码生成器
└── 生成验证码-命令行.exe          # 命令行版验证码生成器
```

---

## 📋 打包前准备

### 1. 安装依赖
确保已安装所有必需的Python包：
```bash
pip install -r requirements.txt
```

### 2. 安装PyInstaller
```bash
pip install pyinstaller
```

### 3. 清理旧文件
打包脚本会自动清理，但也可以手动清理：
```bash
rmdir /s /q dist
rmdir /s /q build
del *.spec
```

## ⚙️ 打包配置说明

### 主程序配置
```python
pyinstaller --onefile ^           # 打包成单个exe
    --windowed ^                   # 无控制台窗口
    --name="企业微信批量发送工具" ^  # exe文件名
    --add-data "wecom;wecom" ^    # 包含wecom模块
    --hidden-import=PIL ^         # 隐藏导入PIL
    --hidden-import=hmac ^        # 隐藏导入hmac（TOTP需要）
    app.py                         # 入口文件
```

### 验证码生成器（GUI）配置
```python
pyinstaller --onefile ^
    --windowed ^                   # 图形界面，无控制台
    --name="获取验证码" ^
    --add-data "wecom;wecom" ^
    --hidden-import=hmac ^
    password_generator_gui.py
```

### 验证码生成器（CLI）配置
```python
pyinstaller --onefile ^
    --console ^                    # 保留控制台窗口
    --name="生成验证码-命令行" ^
    --add-data "wecom;wecom" ^
    --hidden-import=hmac ^
    password_generator.py
```

---

## 🎯 关键参数说明

| 参数 | 说明 | 用途 |
|------|------|------|
| `--onefile` | 打包成单个exe | 方便分发 |
| `--windowed` | 无控制台窗口 | GUI程序使用 |
| `--console` | 保留控制台窗口 | 命令行程序使用 |
| `--name` | 指定exe名称 | 自定义文件名 |
| `--add-data` | 添加数据文件 | 包含模块和资源 |
| `--hidden-import` | 隐藏导入 | 解决动态导入问题 |

## ⚠️ 常见问题

### 1. 打包后运行提示缺少模块
**原因**：某些模块未被自动检测  
**解决**：在打包命令中添加 `--hidden-import=模块名`

### 2. exe文件过大
**原因**：PyInstaller会打包所有依赖  
**说明**：这是正常现象，首次运行后会缓存

### 3. 杀毒软件误报
**原因**：打包的exe可能被识别为可疑  
**解决**：
- 添加到杀毒软件白名单
- 使用代码签名证书（生产环境）
- 告知用户这是正常现象

### 4. 首次运行缓慢
**原因**：需要解压临时文件  
**说明**：第二次运行会快很多
